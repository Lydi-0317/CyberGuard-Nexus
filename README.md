# CyberGuard Nexus

Unified real-time cyber scam detector for SMS, emails, links, WhatsApp/Instagram, bank/UPI frauds. Multi-platform: Web Dashboard, Mobile App, Chrome Extension. Uses ML/DL with live data feeds.

## 🚀 Features
- Multi-modal detection (text, image, tabular, URLs)
- Real-time APIs + user reports + RBI/DoT feeds
- Adaptive models for new 2026 scams

## 🛠️ Tech Stack
- Frontend: React, React Native, Chrome Extension
- Backend: FastAPI, Express
- ML: Hugging Face, XGBoost, PyTorch
- DB: MongoDB, Redis, Pinecone
- Deploy: Docker, GCP/Render
